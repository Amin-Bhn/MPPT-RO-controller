# MPPT-based RO Modules Controller

Arduino source code for an MPPT-based controller driving 10 reverse-osmosis modules.

## Hardware Requirements
- Arduino Uno/Nano
- Voltage divider for Voc measurement (R1=10k, R2=1.43k)
- LM35 temperature sensor
- 10 relays (or LEDs for testing) on pins 2–11

## File
- `main.cpp`: Complete operational code
